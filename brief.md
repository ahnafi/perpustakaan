# aplikasi perpustakaan

## brief

menggunakan mysqli untuk koneksi ke database, register & login menggunakan session, dan dashboard user & dashboard admin.

landing page pada file index.php 

landing page admin pada file dashboard_admin.php

landing page user pada file dashboard_user.php

Buat tampilan sebagus mungkin. interaktif pakai bootstrap v5 yang sudah saya siapkan pada file css/bootstrap.min.css & js/bootstrap.bundle.min.js

ketika register, user langsung diberikan role 'user'

admin tidak perlu register 

admin langsung login dengan email 'admin@example.com' & password 'password' yang sudah dibuat pada database.sql

## fitur 

- login
- register
- logout
- dashboard admin
- dashboard user
- create book
- update book
- delete book
- detail book
- list book
- list category
- create category
- update category
- delete category

- dashboard user & dashboard admin
